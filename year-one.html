1.1 Central Processing Unit (CPU)

  • Executes instructions.
  • Arithmetic Logic Unit (ALU): Performs calculations.
  • Control Unit (CU): Directs data flow.
  • Registers: Fast storage for immediate data.

1.2 Memory & Storage

  • RAM: Temporary data storage.
  • Cache: Faster access to frequently used data.
  • ROM: Stores firmware (BIOS).
  • SSDs & HDDs: Permanent storage.

1.3 Instruction Set Architecture (ISA)

  • Defines how a CPU processes commands.
  • RISC: Simple, fast instructions (ARM).
  • CISC: Complex operations (Intel x86).

2.1 Types of Databases

  • Relational Databases (SQL): MySQL, PostgreSQL.
  • NoSQL Databases: MongoDB, Firebase.
  • Key-Value Stores: Redis.

2.2 Structured Query Language (SQL)

  • SELECT: Retrieves data.
  • INSERT: Adds records.
  • UPDATE: Modifies records.
  • DELETE: Removes records.

3.1 Programming Paradigms

  • Imperative: Step-by-step (C, Python).
  • OOP: Objects & Classes (Java, Python).
  • Functional: Uses functions (Lisp, Haskell).

3.2 Control Structures

  • Conditional Statements: if, else.
  • Loops: for, while.

4.1 Discrete Mathematics

  • Boolean Algebra: Logic gates.
  • Sets & Relations: Database fundamentals.

4.2 Graph Theory

  • Nodes & Edges: Represent relationships.
  • Shortest Path Algorithms: GPS routing.
